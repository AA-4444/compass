# Компас счастья - Design Concept

## Design Philosophy
**"Structured Optimism"** - A modern, energetic interface that reflects the systematic nature of Happi10's "Architecture of Happiness" while maintaining an uplifting, accessible feel.

## Color Palette
- **Primary Yellow:** #FFD700 (Optimism, Energy, Clarity)
- **Primary Red:** #E60000 (Passion, Commitment, Strength)
- **Primary Orange:** #FF4500 (Warmth, Growth, Transformation)
- **Neutral Light:** #F5F5DC (Cream background, approachable)
- **Neutral Dark:** #2C2C2C (Text, contrast)
- **Accent White:** #FFFFFF (Clean, breathing space)

## Typography System
- **Display Font:** Bold, modern sans-serif (e.g., "Montserrat" or "Poppins" Bold 700)
- **Body Font:** Clean, readable sans-serif (e.g., "Inter" or "Poppins" Regular 400-500)
- **Hierarchy:** Large display for title, medium for questions, small for descriptions

## Layout Paradigm
- **Hero Section:** Full-width with circular element (compass/circle motif)
- **Quiz Flow:** Card-based, one question per screen with smooth transitions
- **Result Screen:** Large percentage display with circular progress indicator
- **Call-to-Action:** Prominent button linking to Happi10

## Signature Elements
1. **Circular Progress Indicator:** Reflects the 10 elements as a circle (compass metaphor)
2. **Color-Coded Elements:** Each of the 10 elements gets one of the three brand colors (Yellow, Red, Orange)
3. **Smooth Transitions:** Fade/slide animations between questions for fluidity

## Interaction Philosophy
- **Intuitive Navigation:** Clear next/previous buttons, progress indicator
- **Immediate Feedback:** Visual response to selections (color highlight, subtle animation)
- **Encouraging Tone:** Positive, supportive language throughout

## Animation Guidelines
- **Question Transitions:** Fade-in/fade-out (300ms) for smooth flow
- **Button Hover:** Subtle scale (1.05) and shadow increase
- **Progress Update:** Smooth circular progress animation (500ms)
- **Result Reveal:** Staggered animation (number counts up, then circle fills)

## Key Screens
1. **Welcome/Intro:** Title, brief description, "Start Quiz" button
2. **Quiz Questions:** 10 questions (one per element), 4-5 answer options each
3. **Progress Indicator:** Visual circle showing completion
4. **Result Screen:** Percentage score, element breakdown, link to Happi10
5. **Share/CTA:** "Discover your happiness architecture" button to Happi10
